---
title: "S3 Bucket API"
api: "GET /create/service/db/s3/{YOUR_BUCKET_NAME}"
description: "This endpoint creates a new AWS S3 Bucket and an API to communicate with it"
---

### Path

<ParamField path="YOUR_BUCKET_NAME" type="string" required>
  The name that you want to use to reference the S3 bucket you are creating. Must only be alpha numeric with only lower case letters and no special characters except hyphens.
</ParamField>

### Response

<ResponseField name="statusCode" type="number">
  Based off of standard HTTP request guidelines
</ResponseField>

<ResponseField name="body" type="object">
  Information concerning your newly created s3 bucket
</ResponseField>
